NAME

Kernel::System::ExternalFrontendConfig - external interface config functions

DESCRIPTION

All External interface configuration functions.

PUBLIC INTERFACE

BUILD()

Customization of object construction, do not use it directly.

ExternalConfigGet()

Get home config.

    my %Result = $ExternalFrontendConfigObject->ExternalConfigGet(
        Key   => 'config_key',                # (required) config key
    );

Result: %Result = ( Key => 'config_key', Value => 'value', CreateTime => '2018-07-07 12:00:00', CreateBy => 1, ChangeTime => '2018-07-07 12:00:00', ChangeBy => 1, );

ExternalConfigKeysGet() Set home config.

    my @Keys = $ExternalFrontendConfigObject->ExternalConfigKeysGet(
        Filter => 'HomePage',           # (optional) All keys that starts with.
    );

Result: @Keys = ( 'HomePage::en', 'HomePage::de', ... );

ExternalConfigSet()

Set home config.

    my $Success = $ExternalFrontendConfigObject->ExternalConfigSet(
        Key   => 'config_key',                # (required) config key
        Value => '',                          # (required) Value
        UserID => 1,                          # (required) UserID
    );

ExternalConfigDelete()

Delete home config.

    my $Success = $ExternalFrontendConfigObject->ExternalConfigDelete(
        Key   => 'config_key',                # (required) config key
    );
