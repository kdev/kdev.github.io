NAME

Kernel::WebApp::Plugin::1120DebuggerSTDERRLog - STDERR debugger.

PUBLIC INTERFACE

register()

This plugin collects data sent to STDERR, logs it to the web server log file, and sends it back to the client via custom HTTP header (only if the client requested the sending via X-OTRS-API-Debug header).

To enable this, please activate enable_debugger_stderr_log in WebApp.conf.
