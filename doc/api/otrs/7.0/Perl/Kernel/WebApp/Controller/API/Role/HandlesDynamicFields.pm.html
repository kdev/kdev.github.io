NAME

Kernel::WebApp::Controller::API::Role::HandlesDynamicFields - Role that provides methods to easier deal with dynamic fields.

PUBLIC INTERFACE

CustomerDynamicFieldConfigGet()

This method returns the dynamic field configuration as an array reference only for the fields that are designed for the customer interface.

    my $DynamicFieldsConfig = $Self->CustomerDynamicFieldConfigGet(
        Valid       => 1,
        ObjectType  => [ 'Ticket', 'Article' ],
        FieldFilter => $Self->_TicketDetailViewConfig()->{FollowUpDynamicField} || {},   # optional
    );
