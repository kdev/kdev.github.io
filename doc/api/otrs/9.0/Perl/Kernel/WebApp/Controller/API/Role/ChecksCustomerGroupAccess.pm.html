PUBLIC INTERFACE

Consume this role to be able to check read and write group access. Internally it uses the role Kernel::WebApp::Controller::API::Role::ChecksGroupAccess that provides its public interface.

PRIVATE INTERFACE

_GetGroups()

Retrieves the user groups that matches the given 'Type' parameter.

    my $Groups = $Self->_GetGroups(
        Type 'ro',  #  or 'rw',
    )

Result:

    $Groups => {
        users => 1,
        admin => 1,
        # ...
    };
