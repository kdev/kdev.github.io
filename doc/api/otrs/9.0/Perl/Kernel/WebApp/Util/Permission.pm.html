CheckPermission()

Checks the permission

    my $Permission = $UtilPermissionObject->CheckPermission(
        GroupSettingName => 'Setting',
        or
        GroupSettingName => [ 'Setting1', 'Setting2' ],

        UserID           => 1,
        Type             => 'ro',
    );

Returns 1 (permission granted) or 0 (permission denied).
