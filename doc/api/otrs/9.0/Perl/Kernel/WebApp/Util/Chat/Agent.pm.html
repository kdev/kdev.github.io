OnlineUserList()

Get online user list.

    my $UserList = $ChatUtil->OnlineUserList(
        UserID        => 1,     # (required) UserID
        Users         => 1,     # (optional) Include agents. Default 1.
        Customers     => 1,     # (optional) Include customers. Default 1.
        NoOutOfOffcie => 1,     # (optional) Remove out of office message from Users. Default 1.
        NoDetailData  => 0,     # (optional) No detail data from the user or customer user Default 0.
    );

GetChatStatusForAgent()

Retrieves the chat online status for a given agent.

    my $ChatStatus = $ChatUtil->GetChatStatusForAgent(
        UserID         => 123,  # UserID of the agent
        CustomerUserID => 123,  # UserID of the customer
    );

This method extends the given 'data' hash, therefore it always returns 1.
