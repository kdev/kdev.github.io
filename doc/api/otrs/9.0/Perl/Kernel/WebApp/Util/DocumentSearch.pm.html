NAME

Kernel::WebApp::Util::DocumentSearch - Helper class for the document search functionality.

PUBLIC INTERFACE

has DocumentSearchTypeObjects

Attribute that holds the instantiated document search type objects. The different object modules are located below Kernel::WebApp::Util::DocumentSearch::*, please refer to the single modules for more information about their usage.

has DocumentSearchTypeObjects

Attribute that holds the instantiated document search type objects. The different object modules are located below Kernel::WebApp::Util::DocumentSearch::*, please refer to the single modules for more information about their usage.

has DocumentSearchDriverList

Attribute that holds the list of available document search drivers.

has IndexSearchObject

Attribute that holds the index search object.

Search()

The /Search() method should return the documents based on the given query.

DocumentSearchTypeResultFields()

The /DocumentSearchTypeResultFields() method returns the required fields for the different document search types or only for the given document search type.

DocumentSearchTypeMergeFixedLanguageFilter()

The /DocumentSearchTypeMergeFixedLanguageFilter() method should merge the given language filter to the given predefined filters if needed.

TransformDocumentData()

The /TransformDocumentData() method transforms the given document data for the frontend.

HasFixedLanguageFilter()

The /HasFixedLanguageFilter() method returns the information if the given document type has a fixed language filter.

ChildDocumentSearchTypes()

The /ChildDocumentSearchTypes() method returns the child document search type for the given document search type.

ValidateQuery()

The /ValidateQuery() method checks if the query is correct and complete.

PRIVATE INTERFACE

_LoadDocumentSearchTypeObject()

The /_LoadDocumentSearchTypeObject() method loads the object for the given document search type.

_RemoveEmptyDocumentSearchTypes()

The /_RemoveEmptyDocumentSearchTypes() method removes empty document types from the search result.

_CombineDocumentSearchTypes()

The /_CombineDocumentSearchTypes() method combines document types.

_BuildDocumentSearchDriverList()

The /_BuildDocumentSearchDriverList() method retrieves the list of available document search drivers.

_BuildIndexSearchObject()

The /_BuildIndexSearchObject() method retrieves the index search object.

_BuildDocumentSearchTypeModuleList()

The /_BuildDocumentSearchTypeModuleList() method provides the list of available document search util type modules.
