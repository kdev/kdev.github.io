Cols()

The width (in characters) for this field in the edit mode. It's an alias for the 'Cols' key defined in the field config.

    # Get
    my $Cols = $DynamicField->Cols();

    # Set
    $DynamicField->DefaultValue( 40 );

Rows()

The height (in lines) for this field in the edit mode. It's an alias for the 'Rows' key defined in the field config.

    # Get
    my $Rows = $DynamicField->Rows();

    # Set
    $DynamicField->DefaultValue( 2 );
