DefaultValue()

Returns the default value defined for the field. It's an alias for the 'DefaultValue' key defined in the field config.

    # Get
    my $Default = $DynamicField->DefaultValue();

    # Set
    $DynamicField->DefaultValue( Value => 'some text...', );

RegExList()

Possible list of regular expressions that ensures the text is valid. The regex will be executed with the modifiers xms.

    # Get
    my $RegExList = $DynamicField->RegExList();

    # Set
    $DynamicField->RegExList( Value => [ '^[0-9]$', ], );

FilterFormFieldSchema()

Returns the field schema to use in the form objects.

    my $FieldSchema = $Driver->FilterFormFieldSchema(
        Name         => '...', # optional
        Mandatory    => '...', # optional
        Label        => '...', # optional
        Description  => '...', # optional
        Hint         => '...', # optional
        Placeholder  => '...', # optional
        DefaultValue => '...', # optional
        RequestData => {}, # Mandatory, an empty hash ref as minimum
    );
