PUBLIC INTERFACE

ObjectMatch()

Checks if the provided value is or exists in the object (Ticket, FAQ, etc.) current's dynamic field value.

    my $Match = $DynamicField->ObjectMatch(
        ObjectAttribute => {
            DynamicField_Test  => '...',
            DynamicField_Test2 => [...],
        },
        Value => '...',
    );

Returns

    1 - value matches
    0 - value doesn't match
