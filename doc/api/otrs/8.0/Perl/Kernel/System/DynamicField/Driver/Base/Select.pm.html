DefaultValue()

Returns the default value defined for the field. It's an alias for the 'DefaultValue' key defined in the field config.

    # Get
    my $Default = $DynamicField->DefaultValue();

    # Set
    $DynamicField->DefaultValue( Value => 'key-1', );

PossibleValues()

Returns the possible values for the field. Takes into account the field definition for 'PossibleNone.

    # Get
    my $PossibleValues = $DynamicField->PossibleValues();

    # Get overriding the field definition for 'PossibleNone'.
    my $PossibleValues = $DynamicField->PossibleValues(
        PossibleNone      => 1,
        WithHistoryValues => 1,
    );

    # Set
    $DynamicField->PossibleValues( Value => { 'key-1' => 'Label 1', }, );

PossibleNone()

Indicates if the field supports or has an empty option. It's an alias for the 'PossibleNone' key defined in the field config.

    # Get
    my $PossibleNone = $DynamicField->PossibleNone();

    # Set
    $DynamicField->PossibleNone( Value => 1, );

TranslatableValues()

Indicates if the field options/values are translatable. It's an alias for the 'TranslatableValues' key defined in the field config.

    # Get
    my $TranslatableValues = $DynamicField->TranslatableValues();

    # Set
    $DynamicField->TranslatableValues( Value => 1, );

TreeView()

Indicates if the field should present the values as tree. It's an alias for the 'TreeView' key defined in the field config.

    # Get
    my $TreeView = $DynamicField->TreeView();

    # Set
    $DynamicField->TreeView( Value => 1, );
