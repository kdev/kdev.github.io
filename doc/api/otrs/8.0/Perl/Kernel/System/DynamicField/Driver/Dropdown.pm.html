Link()

Optional HTTP link for the field value in Overviews and detail screens. It's an alias for the 'Link' key defined in the field config.

    # Get
    my $Link = $DynamicField->Link();

    # Set
    $DynamicField->Link( Value => 'https://otrs.com' );

LinkPreview()

This URL will be used for a preview which is shown when this link is hovered in ticket detail. Please note that for this to work, the 'Link' property also needs to be defined, too. It's an alias for the 'LinkPreview' key defined in the field config.

    # Get
    my $LinkPreview = $DynamicField->LinkPreview();

    # Set
    $DynamicField->LinkPreview( Value => 'https://otrs.com' );
