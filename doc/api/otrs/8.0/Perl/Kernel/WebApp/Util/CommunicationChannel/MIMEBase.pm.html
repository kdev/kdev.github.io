Transform()

transform provided data according to the endpoint needs.

    my $Success = $CommunicationChannelObject->Transform(
        Ticket         => {                                              (required) Ticket data
            ...
        },
        Article        => {                                              (required) Article data
            ...
        },
        Context        => 'article',                                     (optional) In case of 'article' context, it builds HTML body if needed
        AttachmentLink => "/api/agent/ticket/1/article/1/attachment/",   (optional) If provided, replace inline attachment links
        ShowExternalImages => 1,                                         (optional)
    );

Returns

    $Success = 1;       # || 0

TransformDataForList()

Transform some article data for the output in the list (directly in the given article data).

    my $UpdateCheckResult = $CommunicationChannelObject->TransformDataForList(
        Data => {},
    );

Returns

    C<undef> - in case an error occurred.
    1 - in case of success.
