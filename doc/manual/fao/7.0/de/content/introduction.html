Einführung¶

Dieses Handbuch listet alle Feature Add-ons auf.

Bemerkung

Die Feature Add-ons werden vom Customer Solution Team in OTRS installiert. Bei On-Premise-Systemen kann der Kunde die Feature Add-ons mit dem Paketmanager installieren, wenn das Customer Solution Team die ausgewählten Feature Add-ons zum Repository hinzugefügt hat. Um ein Feature Add-on zu installieren, wenden Sie sich bitte an das Customer Solution Team über support@otrs.com oder im OTRS Portal.
