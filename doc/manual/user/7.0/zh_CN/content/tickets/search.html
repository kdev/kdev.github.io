搜索¶

使用此屏幕可根据各种条件搜索工单。 工单搜索屏幕位于 工单 菜单的 搜索 菜单项中。

搜索对话框

该对话框分为三个部分：模板、正在使用的过滤器 和 其它的过滤器。

模板¶

对话框的这一部分可用于保存搜索模板以供进一步使用。

若要创建一个新的搜索模板：

  1. 点击 创建搜索模板 按钮。
  2. 输入搜索模板的名称。
  3. 添加一些搜索条件。
  4. 点击 运行搜索 按钮。

若要使用已保存的搜索模板：

  1. 从下拉列表中选择一个模板。
  2. 点击 运行搜索 按钮。

搜索模板对话框

如果只有一张工单匹配，则会打开 工单详情。 否则，您将被重定向到 搜索结果 屏幕以查看所有匹配的工单。

若要删除一个已保存的搜索模板：

  1. 点击 删除 按钮。

警告

模板将立即删除，不会确认！

使用 按模板搜索 按钮直接转到 搜索结果 屏幕。 这是搜索模板的永久链接。

搜索结果屏幕

正在使用的过滤器¶

此部分列出了用于搜索的过滤器。

可以在屏幕的 其它的过滤器 部分添加额外的过滤器，并通过单击该字段旁边的 ⊟ 按钮删除过滤器。

其它的过滤器¶

此部分用于添加更多过滤器并选择结果的输出格式。

增加另一个搜索条件
从下拉列表中选择要在搜索中使用的额外属性。
输出格式
选择搜索结果的输出格式。

您可以通过点击 运行搜索 按钮开始搜索。
