PGP密钥¶

安全通信保护您和您的客户。在GDPR(欧盟发布的“一般数据保护法案”)中，有几篇文章明确提到加密是安全和个人数据保护措施之一。虽然在GDPR下加密不是强制性的，但在某些领域它确实是必不可少的。

OTR允许您在需要时通过 S/MIME证书 或 PGP密钥 加密通信。

注解

由于此软件的独立性，此处不介绍加密所需的服务和软件设置。

使用此屏幕向系统添加PGP密钥。PGP管理屏幕可在 通信和通知 组的 PGP密钥 模块中找到。

PGP管理屏幕

管理PGP密钥¶

注解

要在OTRS中使用PGP密钥，必须先激活其设置。

启用PGP支持

若要添加一个PGP密钥：

  1. 点击左侧边栏的 添加PGP密钥 按钮。
  2. 点击*浏览…*按钮来打开文件对话框。
  3. 从文件系统中选择一个PGP密钥文件。
  4. 点击*添加*按钮。

添加PGP密钥屏幕

若要删除一个PGP密钥：

  1. 点击PGP密钥列表中的垃圾桶图标。
  2. 点击 确认 按钮。

删除PGP密钥屏幕

注解

如果系统中添加了多个PGP密钥，请使用搜索框查找特定的PGP密钥。

PGP配置选项¶

Core → Crypt → PGP
