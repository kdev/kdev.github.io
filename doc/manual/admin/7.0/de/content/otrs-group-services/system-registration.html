Systemregistrierung¶

Wenn die kostenlose Testversion oder die Registrierungsfrist abgelaufen ist, sperrt sich das System selbst. In diesen Fällen müssen Sie Ihr System registrieren oder die Registrierung verlängern.

Nutzen Sie diese Ansicht, um Ihr System bei der OTRS Gruppe zu registrieren. Die Ansicht zur Systemregistrierung ist im Modul Systemregistrierung in der Gruppe Dienstleistungen der OTRS Gruppe verfügbar.

Nicht registriertes System

Systemregistrierung verwalten¶

So registrieren Sie Ihr System:

  1. Sie benötigen eine OTRS-ID. Registrieren Sie dazu sich im OTRS Portal. Nach der erfolgreichen Registrierung erhalten Sie Ihre OTRS-ID.
  2. Gehen Sie ins Modul Systemregistrierung in der Gruppe Dienstleistungen der OTRS Gruppe und geben Sie Ihre OTRS-ID und Ihr Passwort ein.

Systemregistrierung - OTRS-ID hinzufügen

  1. Klicken Sie auf die Schaltfläche Weiter.
  2. Wählen Sie den Systemtyp und geben Sie eine Beschreibung ein.

Systemregistrierung - Systemtyp wählen

  1. Klicken Sie auf die Schaltfläche Registrieren.
  2. Überprüfen Sie Ihre Registrierung.

Registriertes System

So bearbeiten Sie eine Systemregistrierung:

  1. Klicken Sie in der linken Seitenleiste auf die Schaltfläche Details bearbeiten.
  2. Ändern Sie den ggf. den Systemtyp und/oder die Beschreibung.
  3. Klicken Sie auf die Schaltfläche Aktualisieren.

Systemregistrierung bearbeiten

So zeigen Sie die übertragenen Daten an:

  1. Klicken Sie in der linken Seitenleiste auf die Schaltfläche Übertragene Daten anzeigen.
  2. Überprüfen Sie die Daten im JSON-Format.

Übersicht der übertragenen Daten

Siehe auch

Eine detaillierte Information über die Felder ist in Support-Datenanalyse verfügbar.

So erhalten Sie eine Übersicht über die registrierten Systeme:

  1. Klicken Sie in der linken Seitenleiste auf die Schaltfläche Übersicht der registrierten Systeme.
  2. Melden Sie sich im OTRS Portal. an
  3. Überprüfen Sie die Liste der registrierten Systeme.

So deregistrieren Sie ein System:

  1. Klicken Sie in der linken Seitenleiste auf die Schaltfläche System deregistrieren.

Bemerkung

Sie können Ihr System nicht deregistrieren, wenn Sie STORM powered by OTRS™ verwenden oder einen gültigen Servicevertrag haben.
