Az ügyfél webes felülete

Az ügyfeleknek különálló webes felületük van az OTRS-ben, amelyen keresztül új fiókokat hozhatnak létre, megváltoztathatják a fiókbeállításaikat, jegyeket hozhatnak létre és szerkeszthetik azokat, áttekintést kaphatnak azokról a jegyekről, amelyeket létrehoztak, stb.

Folytatva a fenti példát, az ügyfél bejelentkező képernyője a http://www.pelda.hu/otrs/customer.pl URL használatával érhető el egy webböngészővel (lásd a lenti ábrát).
