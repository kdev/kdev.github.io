Core → PDF

PDF::LogoFile

ページ・ヘッダーにおいて、ロゴのためのファイルへのパスを特定します(gif|jpg|png, 700 x 100 pixel)。

この設定は無効化することはできません。

デフォルト値: 

--- <OTRS_CONFIG_Home>/var/logo-otrs.png

PDF::MaxPages

PDFファイルごとの最大ページ数を定義します。

この設定は無効化することはできません。

デフォルト値: 

--- '100'

PDF::PageSize

PDFページの標準サイズを定義します。

この設定は無効化することはできません。

デフォルト値: 

--- a4

PDF::TTFontFile###Monospaced

PDFドキュメントで、等幅フォントを扱うための、パスおよびTTF-Fileを定義します。

この設定は無効化することはできません。

デフォルト値: 

--- DejaVuSansMono.ttf

PDF::TTFontFile###MonospacedBold
