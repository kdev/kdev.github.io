Ticket::Frontend::MenuModule###000-Back

在服务人员界面工单详情视图中，为“返回”菜单显示一个链接。可以通过使用键“Group（组）”和值如“rw:group1;move_into:group2”来实现显示/不显示这个链接的额外访问控制。 为了给菜单条目分组，使用键"ClusterName（菜单组名称）"，其值可为您想在界面上看到的任何菜单组名称（系统默认为Miscellaneous-杂项），使用键"ClusterPriority（菜单组优先级）"来配置工具栏中菜单组的显示顺序。

默认值: 

---
Action: ''
ClusterName: ''
ClusterPriority: ''
Description: Go back
Link: '[% Env("LastScreenOverview") %];TicketID=[% Data.TicketID | html %]'
Module: Kernel::Output::HTML::TicketMenu::Generic
Name: Back
PopupType: ''
Target: ''

Ticket::Frontend::MenuModule###550-Appointment

在服务人员界面工单详情屏幕，在菜单上显示一个链接，以创建一个日历预约并直接链接到此工单。可通过键“Group”和值如“rw:group1;move_into:group2”进行额外的访问控制，以显示或不显示这个链接。如果要放到菜单组中，使用键“ClusterName”，值可以是界面上能看到的任意名称。使用键“ClusterPriority”来配置菜单组在工具栏中显示的顺序。

默认值: 

---
Action: AgentAppointmentCalendarOverview
ClusterName: Miscellaneous
ClusterPriority: '800'
Description: Create a new calendar appointment linked to this ticket
Link: Action=AgentAppointmentCalendarOverview;Subaction=AppointmentCreate;PluginKey=0100-Ticket;ObjectID=[%
  Data.TicketID | uri %]
Module: Kernel::Output::HTML::TicketMenu::Generic
Name: New Appointment
PopupType: ''
Target: ''
