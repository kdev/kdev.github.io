Ticket::Frontend::AgentTicketLockedView###DefaultColumns

可以在服务人员界面锁定的工单视图中过滤的列。 注意：只允许使用工单属性、动态字段（DynamicField_NameX）和客户属性（例如CustomerUserPhone，CustomerCompanyName，...）。

Ticket::Frontend::AgentTicketLockedView###Order::Default

定义服务人员界面锁定的工单视图的默认工单顺序。上：最老的在最上面，下：最近的在最上面。

不能使这个设置失效。

默认值: 

--- Up

Ticket::Frontend::AgentTicketLockedView###SortBy::Default

定义服务人员界面锁定的工单视图工单排序的默认工单属性。

不能使这个设置失效。

默认值: 

--- Age
