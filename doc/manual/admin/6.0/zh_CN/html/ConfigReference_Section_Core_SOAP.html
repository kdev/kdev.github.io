Core → SOAP

SOAP::Keep-Alive

启用SOAP响应的持久连接头。

不能使这个设置失效。

默认值: 

--- 0

SOAP::Password

定义访问SOAP句柄(bin/cgi-bin/rpc.pl)的密码。

这个设置默认没有激活。

默认值: 

--- some_pass

SOAP::User

定义访问SOAP句柄(bin/cgi-bin/rpc.pl)的用户名。

这个设置默认没有激活。

默认值: 

--- some_user
