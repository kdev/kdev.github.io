Frontend → Base → NavBarModule

Frontend::AdminModuleGroups###001-Framework

Legt verfügbare Gruppen für die Administrator-Übersicht fest.

Diese Einstellung kann nicht deaktiviert werden.

Standardwert: 

---
Administration:
  Order: '6000'
  Title: Administration
Automation:
  Order: '4000'
  Title: Processes & Automation
Communication:
  Order: '2000'
  Title: Communication & Notifications
Miscellaneous:
  Order: '7000'
  Title: Miscellaneous
OTRSGroup:
  Order: '5000'
  Title: OTRS Group Services
Ticket:
  Order: '1000'
  Title: Ticket Settings
Users:
  Order: '3000'
  Title: Users, Groups & Roles
