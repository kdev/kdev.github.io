OTRS::ITSM предоставляет следующие отчеты для для оценки ключевых показателей в Управлении Изменениями. Все отчеты доступны при использовании встроенного генератора отчетов.

Количество Изменений за заданный период

При заказе отчета можно задавать относительный (т.е. Изменения, за последние х-дней) или абсолютный период (т.е. Изменения с даты 1 до даты 2). Дополнительно, можно указать статус Изменений, включаемых в отчет.

Отчеты можно получать в "CSV" или "Print" (PDF) форматах.
