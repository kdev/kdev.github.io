导入/导出¶

该软件包提供了一个工具作为通用框架，以 CSV 或 XML 等多种格式导入和导出工单或配置项等对象。

注解

要使用此模块，您需要安装 ITSM变更管理 或为要导入和导出的对象提供后端的任何其他软件包。
