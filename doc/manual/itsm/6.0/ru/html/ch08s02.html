Схема Управления Изменениями в OTRS::ITSM

Реализация Управления Изменениями в OTRS::ITSM определяет Изменение как изменение существующей ИТ инфраструктуры, например, установка нового почтового сервера.

Изменение, обычно, состоит из нескольких подзаданий, OTRS::ITSM позволяет задать любое количество подзаданий на Изменение. Это называют Work Orders/Задачами.
