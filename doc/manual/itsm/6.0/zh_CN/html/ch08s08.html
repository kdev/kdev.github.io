OTRS::ITSM提供了多种方式来查看系统中保存的变更。可以根据需要禁用到这些视图的访问。

变更概览

在一个概览视图窗口中显示所有的变更，可以对每列按升序或降序排序。

图 8.19. 变更-概览


The columns can be defined via SysConfig "Frontend -> Agent -> View -> ITSMChangeOverview -> ITSMChange::Frontend::AgentITSMChange###ShowColumns" with the following available attributes:

表 8.9.  变更概览

属性 激活 详情
实际开始时间 否 实施变更开始的日期和时间
实际结束时间 否 实施变更结束的日期和时间
目录 否 变更分类或类型
变更创建人 是 变更创建人的名字
变更经理 是 变更经理的名字
变更编号 是 系统生成的变更编号
变更状态 是 变更状态
变更状态信号 是 用红绿灯显示的变更状态指示器
变更标题 是 变更的名称
创建时间 否 创建变更时的日期和时间
影响 否 变更的预期效果
计划开始时间 是 计划实施变更的开始日期和时间
计划结束时间 是 预计实施变更的结束日期和时间
优先级 是 变更的优先级别
申请时间 否 客户期望的实施日期
服务 是 变更影响的服务
工作指令数 是 变更相关的工作指令数量

Additionally, the Change Overview can filter and display changes according to various attributes. The following filters are available and can be defined via SysConfig "Frontend -> Agent -> View -> ITSMChangeOverview -> ITSMChange::Frontend::AgentITSMChange###Filter::ChangeStates":

表 8.10.  变更概览过滤器

属性 详情
已请求 显示状态为“已请求”的所有变更
待审批 显示状态为“待审批”的所有变更
被拒绝 显示状态为“被拒绝”的所有变更
通过审批 显示状态为“通过审批”的所有变更
处理中 显示状态为“处理中”的所有变更
成功 显示状态为“成功”的所有变更
失败 显示状态为“失败”的所有变更
取消 显示状态为“取消”的所有变更
撤回 显示状态为“撤回”的所有变更

模版

这个视图显示系统中已定义的模板。服务人员可以点击列标题来按升序或降序排序。

图 8.22. 变更-模板


The columns to be displayed can be defined via SysConfig "Frontend -> Agent -> View -> ITSMChangeTemplateOverview -> ITSMChange::Frontend::AgentITSMTemplateOverview###ShowColumns":

表 8.13.  模版

属性 激活 详情
修改人 否 最后修改模板的服务人员姓名
修改时间 否 最后修改的日期和时间
注释 是 模板的注释或描述
创建人 是 创建模板的服务人员姓名
创建时间 是 创建模板时的日期和时间
删除 是 删除选定模板的选项
名称 是 模板的名称
模板编号 否 模板的内部数据库标识
类型 是 模板的类型
有效性 是 定义模板的有效性（有效、无效和临时无效）。变更创建人不能使用无效或临时无效的模板。

After installing the Change Management module, the following template types are available in the system. These can be defined via SysConfig "Frontend -> Agent -> View -> ITSMChangeTemplateOverview -> ITSMChange::Frontend::AgentITSMTemplateOverview###Filter::TemplateTypes":

表 8.14.  模板类型

属性 详情  
ITSM变更 变更模板  
ITSM工作指令 工作指令模板  
CAB 变更咨询委员会模板  

搜索

为查找适合指定搜索条件的变更或工作指令，系统基于下面的搜索查询属性使用了一个单独的搜索功能：

图 8.23. 变更-搜索


表 8.15.  模版

属性 详情
变更编号 按变更编号搜索，可以使用星号“*”作为通配符。
变更标题 按变更标题或名称搜索
工作指令标题 按工作指令的标题或名称搜索
CAB服务人员 按定义为CAB成员的服务人员搜索
CAB客户 按定义为CAB成员的客户联系人搜索。在这里输入时,自动完成功能提供CAB客户联系人候选的建议。
变更描述 在变更描述文本中搜索
变更理由 在变更的原因/理由中搜索
工作指令说明 在工作指令说明中搜索
工作指令报告 在工作指令报告中搜索
变更优先级 按变更优先级别搜索
变更影响 按变更效果/影响级别搜索
变更类别 按变更类型或类型搜索
变更状态 按变更状态搜索
变更经理 按变更经理搜索
变更创建人 按变更创建人搜索
创建变更的服务人员 按创建这个变更的服务人员用户名搜索
工作指令状态 按工作指令状态搜索
工作指令服务人员 按输入的工作指令服务人员搜索
客户请求日期 按请求变更实施日期的客户搜索。可以按绝对时间（如从日期1到日期2）或相对时间（如在最近X天/周内创建的变更）进行搜索。
计划的开始时间 按计划实施变更的开始日期和时间搜索。可以按绝对时间（如从日期1到日期2）或相对时间（如在最近X天/周内创建的变更）进行搜索。
计划结束时间 按实施变更预计的结束日期和时间搜索。可以按绝对时间（如从日期1到日期2）或相对时间（如在最近X天/周内创建的变更）进行搜索。
实际开始时间 按实施变更实际的开始日期和时间搜索。可以按绝对时间（如从日期1到日期2）或相对时间（如在最近X天/周内创建的变更）进行搜索。
实际结束时间 按实施变更实际的结束日期和时间搜索。可以按绝对时间（如从日期1到日期2）或相对时间（如在最近X天/周内创建的变更）进行搜索。
创建时间 按创建变更的日期和时间搜索。可以按绝对时间（如从日期1到日期2）或相对时间（如在最近X天/周内创建的变更）进行搜索。
修改时间 按修改变更的日期和时间搜索。可以按绝对时间（如从日期1到日期2）或相对时间（如在最近X天/周内创建的变更）进行搜索。
