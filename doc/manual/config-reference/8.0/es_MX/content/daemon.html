Daemon¶

Daemon::PID::Path¶

If enabled the daemon will use this directory to create its PID files. Note: Please stop the daemon before any change and use this setting only if <$OTRSHome>/var/run/ can not be used.

Esta configuración no está activa por omisión.

Valor predeterminado:

/opt/otrs/var/run/

Daemon → Log¶

Daemon::Log::DaysToKeep¶

Define el número de días para mantener los archivos de registro del daemon.

Esta configuración no está activa por omisión.

Valor predeterminado:

1

Daemon::Log::RotationType¶

Type of daemon log rotation to use: Choose “OTRS” to let OTRS system to handle the file rotation, or choose “External” to use a 3rd party rotation mechanism (i.e. logrotate). Note: External rotation mechanism requires its own and independent configuration.

Esta configuración no puede ser desactivada.

Valor predeterminado:

otrs

Daemon::Log::STDERR¶

If enabled the daemon will redirect the standard error stream to a log file.

Esta configuración no puede ser desactivada.

Valor predeterminado:

1

Daemon::Log::STDOUT¶

Si está activado, el demonio redirigirá el flujo de salida estándar a un archivo de registro.

Esta configuración no puede ser desactivada.

Valor predeterminado:

0
