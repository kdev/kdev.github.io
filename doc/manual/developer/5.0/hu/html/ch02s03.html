Naplózó mechanizmus

Az OTRS egy olyan naplózó háttérprogrammal érkezik, amely használható az alkalmazás naplózásánál és a hibakeresésnél.

A Log objektum az ObjectManager modulon keresztül érhető el és használható ehhez hasonlóan: 

$Kernel::OM->Get('Kernel::System::Log')->Log(
    Priority => 'error',
    Message => 'Valamire szükség van!',
);
