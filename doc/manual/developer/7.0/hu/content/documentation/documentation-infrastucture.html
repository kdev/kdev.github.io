Dokumentációs infrastruktúra¶

Az OTRS a Sphinx programot használja a kimenetek előállításához. A HTML kimenet a Read the Docs téma használatával kerül előállításra.

Megjegyzés

Az összes kimenet személyre van szabva az összeállító kiszolgálón. Ha a helyi gépén szeretne egy fejlesztői környezetet felállítani a teszteléshez és a dokumentáció írásához, akkor a kimenetek eltérőek lehetnek.
